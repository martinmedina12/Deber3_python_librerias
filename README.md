Librería utilizada: qrcode

La librería qrcode sirve para generar códigos QR de manera rápida y sencilla
utilizando Python. En donde, se puede convertir textos, enlaces web o 
información en imágenes con códigos QR escaneables desde un celular.

Para instalar la librería en la terminal se ingresa el comando:
pip install qrcode[pill]

Para ejecutar el programa se debe ingresar el tipo de formato que se desea
convertir en código QR (texto o enlace web) y se obtiene la imagen con su 
código.

Ejemplo de uso:

1.Crear QR de texto
2. Crear QR de enlace
3. Salir
Entradas:
Seleccione una opción: 1
Ingrese el texto: Hola
Salidas:
Se genera una imagen llamada qr_texto.png con el código QR
